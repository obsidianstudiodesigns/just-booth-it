# Just Booth It! 📸

Marketing site for **Just Booth It!** — a throwback photo booth rental experience.
*Classic vibes. Timeless memories. Every party could use a booth.*

Dark, party/club-styled single-page site with strobe & laser light effects, scroll
animations, an interactive tilt on feature cards, and a working mobile menu.

## Files
- `index.html` / `styles.css` / `script.js` — the live site (served by GitHub Pages)
- `Just-Booth-It.html` — a **self-contained single file** with all images and video
  embedded (double-click to open offline, or email to a client)
- `assets/` — logo, cropped hero video, sliced icons and photo strips
